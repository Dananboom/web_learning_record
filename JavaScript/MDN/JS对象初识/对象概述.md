# 1 对象概述

## 1.1 对象基础

* 对象是一个包含相关数据和方法的集合

* ```js
  //语法
  var objectName = {
    member1Name : member1Value,
    member2Name : member2Value,
    member3Name : member3Value
  }
  ```

* 对象成员的值是任意的，这个例子

  前四个是成员的资料，这个被成为**属性**

  后两个成员是函数，允许对对象的成员做一些操作，被成为函数

  这样的对象被称为对象字面量，手写出来的对象不同于实例化一个对象

  ```js
  var person = {
    name : ['Bob', 'Smith'],
    age : 32,
    gender : 'male',
    interests : ['music', 'skiing'],
    bio : function() {
      alert(this.name[0] + ' ' + this.name[1] + ' is ' + this.age + ' years old. He likes ' + this.interests[0] + ' and ' + this.interests[1] + '.');
    },
    greeting: function() {
      alert('Hi! I\'m ' + this.name[0] + '.');
    }
  };
  ```

## 2 点表示法

* 可以用点表示法访问对象的属性和方法
* 对象的名字表现为一个命名空间(namespace)必须写在前面，然后是一个`.`，然后是想访问的项目，标识可以是简单的属性的名字，或者是数组属性的一个子元素，又或者是**对象方法的调用**

### 2.1 子命名空间

可以用一个对象来实作另一个对象成员的值，

比如上面的name成员

```js
name : ['Bob', 'Smith'],		
```

改成

```js
name : {
  first : 'Bob',
  last : 'Smith'
},
```

用链式的点表示法，访问

```js
person.name.first
person.name.last
```

**注意**：你需要改变你之前的代码，从

```js
name[0]
name[1]
```

改成

```js
name.first
name.last
```

## 3 括号表示法

```js
person['age']
person['name']['first']
```

这样很像数组。事实上，**数组做到了数字到值的映射**，而对象做的是，**字符串到值的映射** 

## 4 设置对象成员

```js
person.age = 45
person['name']['last'] = 'Cratchit'
```

也可以创建新的成员

```s
person['eyes'] = 'hazel'
person.farewell = function() { alert("Bye everybody!") }
```

### 4.1 括号表示法的特殊之处

可以接收一个变量作为属性的名字，也就是说，成员的名字是可以动态改变的，这一点是点表示法做不到的

## 5 this

this指向了当前代码的运行对象，"this"是非常有用的——它保证了当代码的上下文(context)改变时变量的值的正确性
