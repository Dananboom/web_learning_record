就是选择标签用的。

基础选择器

1.标签选择器
把所有一类的标签选择到，但是不能差异化设置。
2.类选择器
单独选一个或者几个标签 在要选的标签价格class属性 特殊使用-多类名
一个标签有多个名字，在class写好多个类名
3.id选择器
#id名字开头，调用时标签里面是id = “名字” id和类选择器的区别，id只能调用一次，id是在唯一的，经常和js使用
4.通配符选择器
选择页面中所有的标签，但是通过id，类，标签选择器改过样式的不在通配符选择器的范围内

总结

基础选择器 作用 特点 使用情况 用法 备注
标签选择器 可以选出来所有的相同标签 不能差异化选择 较多 p {color：red；……} 无
类选择器 可以选出多个标签 根据需求选择 非常多 .nav {color: red;} 无
id选择器 一次只能选一个标签 一个id在一个html文档只能出现一次，被调用一次 一般和js搭配使用 #nav {color ：red；} 无
通配符选择器 选中所有标签，包括html，body等等 选中所有 特殊情况使用 * {color：red；} 被其他基础选择器选过了的话，通配符对其不起效果
