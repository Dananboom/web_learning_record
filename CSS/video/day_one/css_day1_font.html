小结

属性 表示 注意点
font-size 字号 px单位，要带上；谷歌默认是16px
font-family 字体 对body设置
front-weight 字体的粗细 700-加粗；400-normal；用数字不带单位
font-style 倾斜不倾斜 有normal和italic，但是经常用把倾斜的变成不倾斜的？？？有点疑问
front 连写（语法糖） 1.有顺序：front：font-style font-weight font-size/line-height font-family；
注意不能省略size和family
