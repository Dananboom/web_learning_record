羽生结弦，1994年12月7日出生于日本宫城县仙台市，日本花样滑冰男子单人滑选手。

2014年2月，年仅19岁的羽生结弦夺得索契冬奥会金牌，成为亚洲首位冬奥会男子单人滑冠军 [1-2] ；3月，他夺得世界花样滑冰锦标赛冠军，实现了奥运会、世锦赛、大奖赛总决赛的大满贯 [3] 。

2016年，他在世界花样滑冰大奖赛总决赛上夺冠，实现大奖赛总决赛四连冠 [4] 。2017年4月，他再次夺得世界花样滑冰锦标赛冠军 [5] 。2018年2月，他夺得平昌冬奥会花样滑冰男子单人滑冠军，成为66年来第一位蝉联冬奥会男单冠军的花样滑冰选手 [6] 。
