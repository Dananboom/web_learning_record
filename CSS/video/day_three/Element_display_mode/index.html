什么是显示模式
块元素
自己独占一行
可以给高度和宽度
不给宽度就和腹肌宽度一样宽
是一个盒子，里面可以放行或者块级元素
文字类元素不能使用块元素，比如p，h
行内元素
a，span等
一行可以写多个
不能设置宽高
默认宽度是内容的宽度
行元素内只能放文本或其他行内元素
链接里面不能放链接但是a里面放块级元素好，但是给a转换成块级元素比较安全
行内块元素
img，input，td等
一行可以有多个元素，中间有空隙
默认宽度就是他本身的宽度（行内元素的特点）
高度，行高，外边距及内边距都可以控制
