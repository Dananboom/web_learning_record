传统网页布局的三种方式

  • 普通流
  • 浮动
  • 定位
浮动：选择器{float：属性值}
属性值 描述
none 元素不浮动（默认）
left 元素向左浮动
right 元素向右浮动
左青龙
右白虎

浮动特性

  • 浮动元素会脱离标准流
  • 浮动的元素会一行内显示并元素的顶部对齐
  • 浮动元素会有行内块元素的特性

脱标

  • 脱离普通流的控制,移动到主动位置
  • 浮动盒子不再保留原先的位置（让其他的标准流占用）（叠加的效果）

浮动的元素会一行内显示并元素的顶部对齐

浮动元素会有行内块元素的特性

  • 如果块级盒子没有设置宽度，默认宽度就和父级一样宽，但是添加浮动后它的大小由内容来决定
  • 浮动盒子中间没有缝隙，是挨在一起的

浮动元素经常和标准流的父级元素一起搭配使用

注意

  • 一浮全浮

不给父盒子高度？（清除浮动）

原理：闭合浮动
属性值 描述
left 清楚左侧浮动的影响
right 清除右侧浮动带来的影响
both 同时清除左右两侧带来的影响

清除浮动的方法

  • 额外标签法(找到最后一个浮动盒子，在其后设置一个空标签【空标签必须是块级元素】，class调用clear)(要增加很多无意义的标签，结构性差0)
  • 父级添加overflow属性(父级元素加一句overflow：hidden)但无法显示溢出部分
  • 父级添加：after伪元素
    [.clearfix:after { content: ""; display: block; height: 0; clear: both; visibility: hidden; } .clearfix { /*IE6,7专有*/ *zoom: 1; }]
  • 父级添加伪元素
    [.clearfix:before, .clearfix:after { content: ""; display: table; } .clearfix:after { clear: both; } .clearfix { *zoom: 1; }]
