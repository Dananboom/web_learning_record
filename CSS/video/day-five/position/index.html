定位组成

定位=定位模式＋边偏移

1.定位模式

是通过psoition属性来设置的

值 语义
static 静态定位
relative 相对定位
absolute 绝对定位
fixed 固定定位

2.边偏移

边偏移就是定位盒子移动的最终位置，下面是它的属性

边偏移属性 示例 描述
top top：80px 顶部偏移量，定义元素相对于其父元素上边线的距离
bottom bottom:80px 底部偏移量，定义元素相对于其父元素下边线的距离
left left：80px 左侧偏移量，定义元素相对于其父元素左边线的距离
right right：80px 右侧偏移量，定义元素相对于其父元素右边线的距离

各种定位方式

定位方式 意思 语法 注意点
static静态定位 就是默认定位，无定位的意思 选择器{position：static；}
  1. 静态定位是按照标准流特性摆放位置，它没有边偏移
  2. 很少用到
相对定位relative 相对定位是元素移动位置的时候，是相对于它原来的位置来说的（自恋型定位） 选择器{position：ralative；}
  1. 移动的时候参照自己原来的位置
  2. 它移动走了以后他的位置还保留，其他盒子不会占用这个位置，后面的位置仍然以标准流形式排列
  3. 相对定位是没有脱标的，她最典型的用用是给绝对定位当爹的
绝对定位absolute 绝对定位是元素在移动位置的时候，是相对于他的祖先元素来说的（拼爹型） 选择器{position：absolute；}
  1. 如果没有祖先元素或者祖先元素没有定位，则以浏览器为标准定位（document文档）
  2. 如果父亲有定位（哪种都行）就以最近一级的有定位的祖先元素为参考点移动位置。
  3. 绝对定位脱标，他不再占有原先的位置。
  4. 子绝父相 ，子级使用绝对定位，那么父级使用相对定位
固定定位fixed 固定定位是元素固定于浏览器可视区的位置，主要使用场景：可以在浏览页面滚动时元素的位置不会改变 选择器{position：fixed}
  1. 是以浏览器的可视窗口为参照点移动元素的。
  2. 脱标
  3. 小技巧跟随版心对齐
    • 一个小的算法
    • 先让固定定位的盒子left：50%，走到浏览器的可视区的一半的位置
    • 再让固定定位的盒子margin-left：版心一半的距离，多走版心宽度一半的位置
粘性定位 像是相对定位和固定定位的混合 选择器{position：sticky；top：10px}
  1. 以浏览器的可视窗口为参照点移动元素
  2. 占有原先的位置
  3. 必须加边偏移才有效

定位的叠放次序

z-index

选择器{z-index；1；}

数值可以是正整数，负整数，或者是0，默认的就是auto，数值越大，盒子越靠上

如果没有指定，则谁最后写的最大

只有定位的盒子才有这个属性

定位的拓展

1.绝对定位的居中（不能通过margin）
  • 绝对定位的盒子left50%
  • margin负值左面走盒子的一半
2.绝对定位的垂直居中
  • 绝对定位的盒子top%50
  • margintop负值走盒子的一半

特性

  • 行内元素天界绝对定位和固定定位，如果不给宽度或者是高度，默认大小是内容的大小
  • 可以直接设置宽度和高度
  • 脱标的盒子不会触发外边距塌陷
  • 浮动元素只会压住下面标准流的盒子，但是不会压住文字和图片
  • 绝对定位和固定定位会压住文字
