• display
  • visibility可见性
  • overflow

display

  • display：none隐藏
  • display：block，除了转化成块级元素之外，还有显示元素的意思
  • 不再占有原来的位置

visibility

  • visibility：visible可见
  • visibility：visible隐藏
  • 隐藏以后继续占有原来的位置

overflow

  • overflow：visible：默认
  • overflow：auto在需要的时候添加滚动条
  • overflow：hidden不显示超过内容的尺寸
  • overflow：scroll溢出的部分显示滚动条

注意

一般情况下，我们会隐藏多余的，因为他会影响布局

但是有定位的盒子慎用，因为他会隐藏多余的部分
