## The css2.1 processing model

**在css中，翻译浏览器html源文件以下几步**

1. 从语法上翻译源文件，建立个文件树
2. 识别目标的呈现方式(移动端？电脑端？打印？)
3. 检索所有和目标media type有联系的文档
4. 为文档树的每个元素加个值，这个值根据级联和继承的，并且被media type格式化
5. 生成格式化结构，可能比文档数多，也可能比他少。
6. 从格式化结构转化成medium

### the canvas

就是格式化结构被渲染的空间
